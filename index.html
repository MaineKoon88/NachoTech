Procesador (CPU)

El cerebro de tu computadora, responsable de ejecutar instrucciones y procesar datos. Determina en gran medida el rendimiento general del sistema.

Tarjeta Gráfica (GPU)

Componente especializado en el procesamiento de gráficos, esencial para gaming, diseño gráfico y aplicaciones de renderizado.

Memoria RAM

Memoria de acceso aleatorio que almacena temporalmente los datos que el CPU necesita acceder rápidamente, afectando directamente la multitarea y el rendimiento.

Placa Madre

La base que conecta todos los componentes de tu PC, determinando qué hardware es compatible y cómo interactúan los diferentes elementos del sistema.

Almacenamiento

Dispositivos como HDD, SSD y NVMe que almacenan tu sistema operativo, aplicaciones y archivos permanentemente.

Gabinete

La estructura que alberga y protege todos los componentes de tu PC, influyendo en el flujo de aire, la expansión y la estética del sistema.

Fuente de Poder (PSU)

Convierte la energía de la pared en voltajes utilizables por los componentes de tu PC, siendo crucial para la estabilidad y longevidad del sistema.

Sistemas Operativos

El software fundamental que gestiona los recursos del hardware y proporciona servicios para los programas de computadora.

Procesador (CPU)

El CPU (Unidad Central de Procesamiento) es el componente principal de una computadora que realiza las instrucciones de un programa informático mediante la realización de operaciones aritméticas, lógicas, de control y de entrada/salida.

Características clave:

  • Núcleos: Determinan cuántas tareas puede manejar simultáneamente
  • Hilos: Permiten mejor multitarea mediante tecnología como Hyper-Threading
  • Frecuencia: Velocidad de reloj medida en GHz
  • Cache: Memoria ultrarrápida integrada en el CPU
  • Socket: Tipo de conexión con la placa madre

Marcas principales: Intel (Core i3/i5/i7/i9) y AMD (Ryzen 3/5/7/9)

×

Tarjeta Gráfica (GPU)

La GPU (Unidad de Procesamiento Gráfico) es un procesador especializado diseñado para acelerar la creación de imágenes en un búfer de fotogramas destinado a la salida a un dispositivo de visualización.

Características clave:

  • VRAM: Memoria dedicada para almacenar texturas y datos gráficos
  • Ancho de banda: Velocidad de transferencia de datos de la memoria
  • CUDA Cores/Stream Processors: Unidades de procesamiento paralelo
  • Clock Speed: Velocidad de operación del chip gráfico
  • TDP: Consumo energético y requisitos de refrigeración

Marcas principales: NVIDIA (GeForce RTX) y AMD (Radeon RX)

×

Memoria RAM

La RAM (Memoria de Acceso Aleatorio) es una forma de almacenamiento de datos de computadora que almacena datos y código de máquina que se están utilizando actualmente.

Características clave:

  • Capacidad: Cantidad total de memoria (8GB, 16GB, 32GB, etc.)
  • Tipo: DDR3, DDR4, DDR5 (cada generación mejora velocidad y eficiencia)
  • Velocidad: Medida en MHz, afecta el ancho de banda
  • Latencia (CL): Tiempo de respuesta, menor es mejor
  • Canales: Configuración dual/triple/quad channel mejora rendimiento

Marcas principales: Corsair, G.Skill, Kingston, Crucial, Team Group

×

Placa Madre

La placa base es la tarjeta de circuito impreso principal que contiene los conectores para el CPU, RAM, tarjetas de expansión y proporciona conectividad para todos los demás componentes del sistema.

Características clave:

  • Factor de forma: ATX, Micro-ATX, Mini-ITX (determina tamaño y características)
  • Socket: Tipo de conexión para el CPU (debe coincidir con el procesador)
  • Chipset: Determina características soportadas y overclocking
  • Ranuras RAM: Cantidad y tipo de memoria soportada
  • Conectividad: USB, SATA, M.2, PCIe, audio, red, etc.
  • VRM: Sistema de alimentación para el CPU

Marcas principales: ASUS, MSI, Gigabyte, ASRock, Biostar

×

Almacenamiento

Los dispositivos de almacenamiento guardan permanentemente el sistema operativo, aplicaciones y archivos personales, incluso cuando la computadora está apagada.

Tipos principales:

  • HDD (Disco Duro): Almacenamiento mecánico, mayor capacidad, menor velocidad
  • SSD SATA: Estado sólido, sin partes móviles, 4-5x más rápido que HDD
  • NVMe M.2: Ultra rápido (hasta 7GB/s), conectado directamente al PCIe

Consideraciones:

  • Capacidad: Desde 256GB hasta múltiples TB
  • Velocidad: Lectura/escritura secuencial y aleatoria
  • Durabilidad: TBW (Terabytes escritos) para SSDs
  • Factor de forma: 2.5", M.2, U.2

Marcas principales: Samsung, Western Digital, Seagate, Crucial, Kingston

×

Gabinete

El gabinete es la estructura que alberga y protege todos los componentes internos de tu computadora, influyendo en la refrigeración, expansión y estética del sistema.

Características clave:

  • Factor de forma: ATX, Micro-ATX, Mini-ITX (debe coincidir con la placa madre)
  • Espacio para refrigeración: Ventiladores, radiadores para refrigeración líquida
  • Administración de cables: Espacio detrás de la placa madre, pasacables
  • Bahías de expansión: Para discos duros, unidades ópticas, etc.
  • Ventilación: Flujo de aire optimizado para mejor refrigeración
  • Vidrio templado: Para mostrar los componentes internos

Consideraciones:

  • Tamaño: Asegúrate que quepa tu GPU y refrigeración
  • Refrigeración: Espacio para ventiladores o radiadores
  • Filtros antipolvo: Mantienen limpios los componentes
  • Conectores frontales: USB, audio, controles de ventilador

Marcas principales: NZXT, Corsair, Lian Li, Cooler Master, Fractal Design

×

Fuente de Poder (PSU)

La fuente de alimentación convierte la corriente alterna (AC) de la pared en corriente continua (DC) que los componentes de la computadora pueden usar, proporcionando voltajes estables y limpios.

Características clave:

  • Potencia: Medida en Watts (500W, 750W, 1000W, etc.)
  • Eficiencia: Certificación 80 Plus (White, Bronze, Silver, Gold, Platinum, Titanium)
  • Modularidad: Totalmente, semi o no modular (para gestión de cables)
  • Conectores: CPU, PCIe, SATA, Molex para alimentar todos los componentes
  • Protecciones: OVP, UVP, OCP, SCP, OTP para seguridad

Consideraciones:

  • Calidad de los componentes internos (capacitores japoneses son mejores)
  • Ventilación y ruido (fan size, modo cero RPM)
  • Garantía (las buenas fuentes ofrecen 7-10 años)

Marcas principales: Seasonic, Corsair, EVGA, be quiet!, Thermaltake

Sistemas Operativos

El sistema operativo es el software más importante que se ejecuta en una computadora, gestiona los recursos del hardware y proporciona servicios para los programas de computadora.

Windows

  • Desarrollador: Microsoft
  • Última versión: Windows 11
  • Ventajas: Mayor compatibilidad de software y juegos, interfaz amigable
  • Desventajas: Menor privacidad, susceptible a malware
  • Uso ideal: Usuarios generales, gaming, oficina

macOS

  • Desarrollador: Apple
  • Última versión: macOS Ventura
  • Ventajas: Excelente optimización, diseño pulido, buena seguridad
  • Desventajas: Limitado a hardware Apple, menos personalizable
  • Uso ideal: Diseño gráfico, desarrollo, productividad

Linux

  • Distribuciones populares: Ubuntu, Fedora, Debian, Arch
  • Última versión: Varía por distribución
  • Ventajas: Open-source, altamente personalizable, muy seguro
  • Desventajas: Curva de aprendizaje, menos soporte para algunos softwares
  • Uso ideal: Servidores, desarrollo, usuarios avanzados
