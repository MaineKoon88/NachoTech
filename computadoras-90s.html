Hardware Representativo

Intel 486 (1989-1994)

El procesador que llevó las PCs al mainstream

  • Frecuencia: 25-100 MHz
  • Núcleos: 1
  • Cache: 8KB L1

Intel Pentium (1993)

Primera arquitectura superscalar para consumidores

  • Frecuencia: 60-200 MHz
  • Transistores: 3.1 millones
  • Nota: Famosa por el "bug FDIV"

3dfx Voodoo (1996)

Primer acelerador 3D dedicado para juegos

  • Memoria: 4MB EDO RAM
  • Resolución: Hasta 800x600
  • FPS: ~30 en Quake

Juegos Icónicos y su Hardware Requerido

Doom (1993)

FPS | id Software

Requisitos Mínimos:

  • CPU: 386 a 25 MHz
  • RAM: 4MB
  • Gráficos: VGA
  • Espacio: 10MB (disquetes)

Requisitos Recomendados:

  • CPU: 486 a 66 MHz
  • RAM: 8MB
  • Gráficos: SVGA
  • Nota: Con soundcard para audio digital

Warcraft II (1995)

Estrategia | Blizzard

Requisitos Mínimos:

  • CPU: 486 a 33 MHz
  • RAM: 8MB
  • Gráficos: SVGA
  • Espacio: 20MB (CD-ROM)

Requisitos Recomendados:

  • CPU: Pentium 75 MHz
  • RAM: 16MB
  • Gráficos: Acelerador 2D
  • Nota: Requería Windows 95 para versión PC

Diablo (1996)

ARPG | Blizzard North

Requisitos Mínimos:

  • CPU: Pentium 60 MHz
  • RAM: 8MB
  • Gráficos: SVGA
  • Espacio: 80MB (CD-ROM)

Requisitos Recomendados:

  • CPU: Pentium 90 MHz
  • RAM: 16MB
  • Gráficos: Acelerador 2D
  • Nota: Modo multijugador requería conexión IPX

Configuraciones Típicas por Año

1992

  • CPU: Intel 486SX 25MHz ($800)
  • RAM: 4MB SIMM ($200)
  • Disco: 120MB HDD ($300)
  • Gráficos: Tseng ET4000 1MB ($250)
  • Sistema: MS-DOS 5.0 + Windows 3.1

1996

  • CPU: Pentium 133MHz ($900)
  • RAM: 16MB EDO ($150)
  • Disco: 1.2GB HDD ($250)
  • Gráficos: Matrox Mystique 2MB + Voodoo1 ($600)
  • Sistema: Windows 95 OSR2

1999

  • CPU: Pentium III 450MHz ($700)
  • RAM: 64MB SDRAM ($120)
  • Disco: 8GB HDD ($200)
  • Gráficos: NVIDIA Riva TNT2 32MB ($300)
  • Sistema: Windows 98 SE
